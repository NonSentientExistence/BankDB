# BankDB
MySQL Database for fictive bank

Har skapat en fiktiv bankdatabas då jag tyckte det var intressant att sätta upp, på grund av att jag jobbat inom bank och finans tidigare. 
Har försökt hålla den något så när korrekt utifrån bokföring och hur transaktioner kan genomföras men det saknas givetvis mycket fält, data och funktioner för att vara en fullt funktionell bank databas.
